# 53

**思路：**动态规划

**状态：**dp[i]表示以num[i]为结尾的连续子序列的最大和

**状态转换方程：**dp[i] = Math.max(dp[i - 1] + nums[i], nums[i])

**实例：**

| nums | -2   | 1    | -3   | 4    | -1   | 2    | 1    | -5   | 4    |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| dp   | -2   | 1    | -2   | 4    | 3    | 5    | `6`  | 1    | 5    |
